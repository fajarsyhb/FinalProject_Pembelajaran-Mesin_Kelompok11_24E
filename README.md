# FinalProject_Pembelajaran-Mesin_Kelompok11_24E
Judul : Analisis Sentimen Ulasan Produk E-Commerce Menggunakan Perbandingan Algoritma Support Vector Machine, Random Forest, dan Logistic Regression Berbasis TF-IDF  Dataset : https://www.kaggle.com/datasets/datafiniti/consumer-reviews-of-amazon-products

Di era digitalisasi saat ini, industri e-commerce global seperti Amazon telah menjadi platform utama bagi jutaan konsumen untuk berbelanja sekaligus memberikan umpan balik (feedback) terhadap produk yang mereka beli. Ulasan konsumen (consumer reviews) tersebut merupakan aset data yang sangat berharga. Bagi pihak manajemen, ulasan tersebut menyimpan informasi mengenai kepuasan pelanggan dan evaluasi kualitas produk. Bagi calon pembeli lain, ulasan tersebut menjadi rujukan utama sebelum melakukan transaksi.
Namun, kendala utama yang dihadapi adalah volume ulasan yang masuk setiap harinya berskala sangat besar (big data) dan mayoritas berbentuk teks tidak terstruktur (unstructured text). Konsumen sering kali menulis ulasan menggunakan bahasa kasual, singkatan, salah ketik (typo), serta panjang kalimat yang tidak seragam. Jika proses pemetaan sentimen (apakah ulasan tersebut bermakna positif, netral, atau negatif) dilakukan secara manual oleh tenaga manusia, hal tersebut menjadi tidak efisien, membutuhkan waktu yang lama, dan rawan terhadap subjektivitas.
Oleh karena itu, diperlukan sebuah pendekatan komputasi cerdas berbasis Natural Language Processing (NLP) dan Machine Learning untuk mengklasifikasikan sentimen secara otomatis. Melalui pembobotan kata menggunakan metode Term Frequency-Inverse Document Frequency (TF-IDF), teks ulasan dapat dikonversi menjadi representasi numerik. Penelitian ini difokuskan untuk melakukan studi komparasi terhadap tiga algoritma klasifikasi dengan karakteristik yang berbeda, yaitu Support Vector Machine (SVM) sebagai perwakilan model linear berbasis margin, Random Forest sebagai perwakilan model ensemble pohon keputusan, dan Logistic Regression sebagai model statistik linier dasar. Perbandingan ini krusial untuk menemukan model terbaik yang paling adaptif dan efisien dalam menangani data ulasan skala besar.
